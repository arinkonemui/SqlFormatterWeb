SQL整形ツール 使い方ガイド

SQL整形ツールは、複雑なSQLクエリをブラウザだけで整形・作成できる無料ツールです。 このページでは基本的な使い方を説明します。

基本的な使い方

SQLを入力する

「整形」タブの入力エリアに、整形したいSQLクエリを貼り付けます。

オプションを設定する

必要に応じて、データベース方言・インデントサイズ・大文字/小文字などのオプションを選択します。

「整形」ボタンをクリック

ボタンを押すと、右側（または下部）の出力エリアに整形結果が表示されます。

結果をコピー

「コピー」ボタンで整形結果をクリップボードにコピーできます。

対応データベース方言

以下のデータベースに対応しています：

MySQL PostgreSQL SQLite SQL Server 標準SQL
方言によって予約語やサポートする構文が異なります。正確な整形のために、使用するデータベースに合った方言を選択してください。

CREATE文ビルダー

「CREATE」タブでは、GUIでテーブル定義を入力し、CREATE TABLE文を自動生成できます。

テーブル名を入力

作成するテーブルの名前を入力します。

カラムを追加

カラム名、データ型、制約（NOT NULL、PRIMARY KEY など）を設定します。

SQL を生成

「生成」ボタンで CREATE TABLE 文が生成されます。

注意事項

  • データはブラウザ内でのみ処理されます — サーバーへの送信は一切ありません。
  • 機密情報の入力はお控えください — パスワード、APIキー、個人情報などは入力しないでください。
  • 整形結果の正確性は保証しません — 本番環境で使用する前に、必ずテスト環境で検証してください。
  • 極度に複雑なクエリは正しく整形されない場合があります。
